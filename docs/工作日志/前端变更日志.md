## 9. 当前剩余主要风险

| 风险 | 说明 | 建议 |
| --- | --- | --- |
| 旧 UI 文案乱码 | 多个 Vue 文件中存在历史乱码文本。 | 单独做一次 UTF-8 文案清理，不和功能开发混在一起。 |
| 断面降级冲突提示仍较基础 | 当前已严格阻止会删除已配置车道规则的断面降级，但仍复用顶部错误提示。 | 后续可接入统一 toast/notification 展示冲突详情和定位。 |
| 动态拓扑自愈不足 | 当前道路交叉自动打断、路口多边形重算还未完整实现。 | 作为 FR1 后续重点。 |
| FR2/FR3/FR4/FR6 仍是部分能力 | 节点 Gizmo、车道连接器 UI、严格仿真、飞线与报表仍需持续开发。 | 按《前端开发计划.md》分阶段推进。 |
